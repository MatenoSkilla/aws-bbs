# 📝 Node.js 掲示板アプリ（Docker構成）

このプロジェクトは、**Node.js + Express + MySQL** を使用したシンプルな掲示板APIアプリです。  
**Docker Composeで構築可能**で、将来的には **AWS EC2 + RDS** へ移行を想定しています。

## 📦 使用技術

- Node.js (v18)
- Express
- MySQL
- Docker / Docker Compose
- dotenv
